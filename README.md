# 我的 sliders

自己做的小工具, 可以直接把 markdown 文件转换成 revealjs 的 ppt 文件.

配合 gh-pages 可以很方便的制作 ppt.

## 使用方式

### 克隆项目(可以 fork 到自己的仓库)

    git clone https://github.com/teddy-ma/sliders.git
    cd sliders

### 安装依赖并构建(确保 node 环境已安装)

    npm install
    edit src/xxx.md
    npm run build

### 清理构建

    npm run clean

### 预览

    npm run preview
